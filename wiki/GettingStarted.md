# Getting Started

## Prerequisites

- **C++20 compiler** (e.g., GCC 10+, Clang 12+; MacOS and Linux only)
- Python 3
- CMake

## Build & Run

```bash
# Build using the top-level script (recommended)
python3 beacon-build.py

# Run the system (default: 30 seconds)
python3 beacon-run.py 30
```

See [Apps](Apps.md) for details on each application.
