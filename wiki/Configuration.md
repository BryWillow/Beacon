# Configuration

All configs are in `config/playback/` and referenced from `default.json`.

- **Senders:** UDP, TCP, Console (`senders/`)
- **Authorities:** Burst, Chaos, Wave, Rate Limit, etc. (`authorities/`)
- **Advisors:** Symbol priority, burst detection (`advisors/`)
- **Main config:** `default.json` references all components.

See [Testing](Testing.md) for validation.
